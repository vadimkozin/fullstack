# fullstack first app

## install
```bash
    git clone https://github.com/vadimkozin/fullstack
    cd fullstack
    npm install
    edit config/keys.dev
        module.exports = {
            PORT: input_you_port,
            JWT_SECRET: 'input_you_jwt_secret',
            MONGO_URI: 'input_you_string_connent_to_mongodb'
        }
    run `http://localhost:5000/`
    ( if input_you_port=5000 )
```
